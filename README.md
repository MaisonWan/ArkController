## Ark Controller
Ark Controller是基于adb封装的Android控制器，运行在Windows系统之上。详情查看[GitHub Page](https://maisonwan.github.io/ArkController/) 


### 连接设备
等待设备连接，获取可以连接设备列表，封装命令

```connect
等待设备连接
`adb shell wait-for-device devices`
```
